What is pLitter?

pLitter (Plastic litter) is a standardized, deep learning-friendly image dataset and pre-trained model for roadside plastic litter identification. The dataset includes images of plastic littered scenes and annotations in COCO format. The main features of the dataset are:

  • Object detection
  • 10,000+ images
  • 50,000+ object instances
  • 4 object categories (Facial mask, Pile, Plastic, and Trash bin)

Misson

Developing necessary tools and methods in order to:

  • grow the pLitter dataset while expanding the taxonomy of plastic litter object categories to become a generalized global dataset.
  • create heatmaps from trained models to identify large scale littering trends, especially plastic leakage into water bodies.
