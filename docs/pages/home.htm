What is pLitter?

pLitter (Plastic litter) is a standardized, deep learning friendly image dataset and pre-trained model for plastic litter identification in streets, road-sides, etc. This dataset includes images of plastic littered scenes and annotations in COCO format. The main features of the dataset are,

  • Object detection
  • 10K images
  • 50K object instances
  • 4 object categories

Misson

Developing necessary tools and methods,

  • To grow the pLitter dataset, expand the taxonomy of object categories of plastic litter to adapt globally and eventually build a generalized dataset.
  • To use the trained models to predict the plastic litter on streets and roads on a large scale, to produce the heatmaps to identify and analyse the situation and trends of littering, especially identifying littering hotspots, and plastic leakage into water canals or rivers.
