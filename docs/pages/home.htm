What is pLitter?

pLitter (Plastic litter) is a standardized, deep learning friendly image dataset and pre-trained model for plastic litter identification in streets, road-sides, etc.

  • Object detection
  • 10K images (>6K labeled)
  • 20K object instances
  • 4 object categories
