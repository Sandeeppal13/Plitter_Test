What is pLitter?

pLitter (Plastic litter) is a standardized, deep learning-friendly image dataset and pre-trained model for roadside plastic litter identification. The dataset includes images of plastic littered scenes and annotations in COCO format.

The main features of the dataset are:
  • Object detection
  • 10,000+ images
  • 50,000+ object instances
  • 4 object categories (Facial mask, Pile, Plastic, and Trash bin)

We are using the Citizen Science approach to expand the pLitter taxonomy to more categories as listed below.

  • Bottle (Beverage)
  • Bottle (Other)
  • Carrier bag
  • Clothes or Textile
  • Container
  • Cup
  • Cutleries
  • Diaper
  • Facial mask
  • Fishing gear
  • Foil
  • Pile
  • Plastic (Other)
  • Plate
  • Rope
  • Trash bin
  • Wrapper or Sachet
